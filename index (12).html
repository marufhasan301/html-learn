Basic Structure

HTML
DOCTYPE, html[lang], head(meta,title,link), এবং body — এগুলো অবশ্যই থাকবে। head-এ script/style/meta যায়। 
<!doctype html>
<html lang="bn">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width,initial-scale=1">
    <title>My Page</title>
    <link rel="stylesheet" href="styles.css">
    <meta name="description" content="একটি নমুনা পেজ">
  </head>
  <body>
    <header><h1>হ্যালো</h1></header>
    <main><p>কনটেন্ট...</p></main>
    <footer><small>Footer</small></footer>
    <script src="app.js" defer></script>
  </body>
</html>

Text Content (হেডিং, প্যারাগ্রাফ, ইনলাইন ট্যাগ)

হেডিং, প্যারা, emphasis

হেডিং: <h1>…</h1>. পারা: <p>. গুরুত্বের জন্য <strong>, italic জন্য <em>.

Lists & code

  • Ordered: <ol>
  • Unordered: <ul>

Other important topics (Global attrs, Events, Meta, Web APIs, Web Components, Canvas/SVG)

Global attributes (প্রায় সব ট্যাগে লাগবে)

  • id, class, style
  • data-* (custom data attributes)
  • hidden, title, tabindex, contenteditable
<div id="card" class="card primary" data-user="123" tabindex="0">Content</div>

Event attributes & JS hooks

কিছু সাধারণ ইভেন্ট: onclick, oninput, onchange, onkeydown, onload ইত্যাদি।

<button onclick="doSomething()">Click</button>

<script>
function doSomething(){ alert('action') }
</script>

Meta / Head extras (SEO, performance)

Head-এ রাখার উপকারি ট্যাগ: <meta charset>, viewport, description, <link rel="preload">, manifest।

<meta name="description" content="Site description">
<link rel="preload" href="/fonts/xyz.woff2" as="font" type="font/woff2" crossorigin>
<link rel="manifest" href="/manifest.json">

Web APIs (popular)

  • Fetch API — network requests
  • Canvas / WebGL — drawing
  • Geolocation, MediaDevices (getUserMedia), WebRTC
  • LocalStorage, IndexedDB — storage
  • Service Worker — offline/PWA
// Fetch example
fetch('/api/data').then(r => r.json()).then(data => console.log(data));

// getUserMedia example
navigator.mediaDevices.getUserMedia({video:true,audio:false}).then(stream => { /* show stream */ });

Web Components (Custom elements)

Custom element উদাহরণ (ছোট):

<script>
class MyWidget extends HTMLElement{
  connectedCallback(){ this.innerHTML = '<div>Hello from widget</div>'; }
}
customElements.define('my-widget', MyWidget);
</script>

Deprecated / Avoid
